# SoftwareRasterizer
Software Rasterizer written from scratch in C++. Used as part of the Graphics Programming course.
Both a **build** and the **source code** is available.

## Interesting files
- [Renderer](_PROJECT/Source/EliteRasterizer/ERenderer.cpp): the heart of the rasterizer. Performs the vertex transformation, rasterization and pixel shading.
- [Renderer Helper Functions](_PROJECT/Source/EliteRasterizer/ERenderer.h#L77): some helper functions used by the renderer above.
- [Camera](_PROJECT/Source/EliteRasterizer/ECamera.h): simple camera with Unreal Engine 4 controls.
